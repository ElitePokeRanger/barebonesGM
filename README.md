barebonesGM
===========
Garry's Mod Bare bones Gamemode, planned to have a complete custom HUD, scoreboard, and possibly chat &amp; weapons.

Current Features
===========
-Random rebel model every time you spawn

-A "sudden death" style mode

-Simplistic HUD (WIP)
